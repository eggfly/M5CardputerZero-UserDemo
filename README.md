# M5CardputerZero-UserDemo

[中文](./README_ZH.md)

A user demo application for the **M5Cardputer Zero** device, built on the [M5Stack_Linux_Libs](https://github.com/m5stack/M5Stack_Linux_Libs) SDK. This project demonstrates how to develop a graphical UI application using **LVGL 9.5** on M5Cardputer Zero (AArch64 Linux).

The UI is generated by **SquareLine Studio 1.5.0**, featuring a status bar (Logo, clock, battery) and a main content area. Two display backends are supported: **SDL2** (for PC simulation/debugging) and **Linux Framebuffer** (for running on the device).

## Features

- Graphical UI powered by LVGL 9.5
- Status bar: M5Stack Zero logo, current time, battery percentage
- Main content area: app name + page content placeholder
- Two display backends:
  - **SDL2** — PC simulation for local debugging (default build mode)
  - **Linux Framebuffer (ST7789V)** — runs on the M5Cardputer Zero device
- evdev keyboard/touch input support (on-device)
- SCons + Kconfig build system with `scons menuconfig` for flexible configuration

## Prerequisites

### Install Dependencies (once)

```bash
sudo apt update
sudo apt install python3 python3-pip libffi-dev libsdl2-dev

pip3 install parse scons requests tqdm
pip3 install setuptools-rust paramiko scp
```

> Python version ≥ 3.8 is required.

### Clone the Repository (with Submodules)

```bash
git clone --recursive https://github.com/dianjixz/M5CardputerZero-UserDemo.git
cd M5CardputerZero-UserDemo
```

If you already cloned without `--recursive`:

```bash
git submodule update --init --recursive
```

## Build

### Option 1: SDL2 Simulation Mode (PC, default)

The SDL2 backend is enabled by default in `projects/UserDemo/SConstruct`. Build and run on your PC:

```bash
cd projects/UserDemo
scons -j$(nproc)
```

The compiled binary will be located at `projects/UserDemo/build/M5CardputerZero-UserDemo`.

### Option 2: Cross-Compile for AArch64 (M5Cardputer Zero device)

Install the AArch64 cross-compilation toolchain:

```bash
# Option A: via apt
sudo apt install gcc-aarch64-linux-gnu g++-aarch64-linux-gnu

# Option B: download Linaro toolchain manually
wget https://m5stack.oss-cn-shenzhen.aliyuncs.com/resource/linaro/gcc-linaro-7.5.0-2019.12-x86_64_aarch64-linux-gnu.tar.xz
sudo tar Jxvf gcc-linaro-7.5.0-2019.12-x86_64_aarch64-linux-gnu.tar.xz -C /opt
export PATH="/opt/gcc-linaro-7.5.0-2019.12-x86_64_aarch64-linux-gnu/bin:$PATH"
```

Edit `projects/UserDemo/SConstruct` to switch from SDL2 to Framebuffer backend:

```python
# Disable SDL2 — change "if True" to "if False":
if False:   # <-- changed to False
    ...
    f.write('CONFIG_V9_5_LV_USE_SDL=y\n')

# Enable Framebuffer — change "if False" to "if True":
if True:    # <-- changed to True
    ...
    f.write('CONFIG_V9_5_LV_USE_LINUX_FBDEV=y\n')
    f.write('CONFIG_V9_5_LV_DRAW_SW_ASM_NEON=y\n')
    f.write('CONFIG_TOOLCHAIN_PREFIX="aarch64-linux-gnu-"\n')
```

Then build:

```bash
cd projects/UserDemo
scons -j$(nproc)
```

### Build Management Commands

```bash
# Interactive configuration menu
scons menuconfig

# Clean build artifacts
scons -c

# Full clean (including cached config)
scons distclean

# Verbose build output
scons verbose
```

## Run

### PC (SDL2 Simulation)

```bash
cd projects/UserDemo
# Optionally set window resolution (default: 320x170)
export LV_SDL_VIDEO_WIDTH=320
export LV_SDL_VIDEO_HEIGHT=170
./build/M5CardputerZero-UserDemo
```

### M5Cardputer Zero Device

Copy the binary to the device:

```bash
scp projects/UserDemo/build/M5CardputerZero-UserDemo user@<device_ip>:/home/user/
```

Run on the device (Framebuffer mode):

```bash
# Auto-detect the ST7789V framebuffer device
./M5CardputerZero-UserDemo

# Or manually specify the framebuffer device
export LV_LINUX_FBDEV_DEVICE=/dev/fb0
./M5CardputerZero-UserDemo

# Specify the keyboard input device
export LV_LINUX_KEYBOARD_DEVICE=/dev/input/by-path/platform-3f804000.i2c-event
./M5CardputerZero-UserDemo
```

## UI Layout

The UI is designed in SquareLine Studio 1.5.0 for a **320×170** display (ST7789V).

| Region | Content |
|--------|---------|
| Top-left | M5Stack Zero logo image |
| Top-right (clock) | Clock icon + time label |
| Top-right (battery) | Battery icon + percentage label |
| Top-center | App name label |
| Main area | App page content (320×143) |

To customize the UI, edit the project in SquareLine Studio and regenerate the code under `projects/UserDemo/main/ui/`.
